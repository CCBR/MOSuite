# Differential gene expression analysis for the NIDAP test dataset. The result of running `diff_counts()` on `nidap_filtered_counts`.

Differential gene expression analysis for the NIDAP test dataset. The
result of running
[`diff_counts()`](https://ccbr.github.io/MOSuite/reference/diff_counts.md)
on `nidap_filtered_counts`.

## Usage

``` r
nidap_deg_analysis_2
```

## Format

An object of class `list` of length 3.
