# Function for testing CLI argument parsing

Function for testing CLI argument parsing

## Usage

``` r
do_math(add = TRUE, subtract = FALSE, left = 1, right = 2)
```

## Arguments

- add:

  whether to add left and right

- subtract:

  whether to subtract left and right

- left:

  number on the left side of the operand

- right:

  number on the right side of the operand

## Value

result of adding or subtracting left and right
