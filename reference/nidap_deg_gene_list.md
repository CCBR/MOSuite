# List of differentially expressed genes from the NIDAP test dataset using default parameters with `filter_diff()`.

List of differentially expressed genes from the NIDAP test dataset using
default parameters with
[`filter_diff()`](https://ccbr.github.io/MOSuite/reference/filter_diff.md).

## Usage

``` r
nidap_deg_gene_list
```

## Format

An object of class `data.frame` with 641 rows and 16 columns.
